Azure Dirt

Obtaining

Azure dirt is generated underneath lapis roses. It can be obtained only by mining it with a silk-touch-enchanted tool. Breaking it with any other tool drops one azure splinter.

Usage

Using shears on azure dirt converts the azure dirt to regular dirt and drops one lapis splinter. Nine lapis splinters can be crafted into a lapis lazuli.

Using bonemeal on an azure dirt block with air above it has a 15% chance to grow a new lapis rose.
