Condenser

Obtaining

Condensers can be crafted using two redstone, a dropper, a crafting table, four smooth stone, and a copper ingot.

A condenser drops itself when mined with a pickaxe. Otherwise, it drops nothing.

Usage

A condenser can hold a single stack of input items that has a crafting recipe that turns three or more of the same item into a different item. The condenser can also hold a single stack of output items.

Using the condenser with an empty hand or a stack that matches the output items removes the output item stack. If there are no items, the input items are removed instead.

Using the condenser with a hand full of items that can be condensed puts those items into the condenser's input slot.

While powered with a redstone signal, the condenser crafts input items into outputs at a rate of one recipe use per second. If the condenser is facing a container, crafted items are pushed into the container.

Hoppers on the top and non-front sides of the condenser can insert items into the input slot, and hoppers on the bottom remove items from the output slot.
