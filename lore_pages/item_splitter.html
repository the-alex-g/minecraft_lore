Item Splitter

Obtaining

Item splitters can be crafted using eight smooth stone and a hopper. An item splitter drops itself when mined with a pickaxe, or nothing with any other tool.

Usage

An item splitter holds a single stack of items that it evenly distributes between all adjacent containers except for other item splitters and hoppers pointing into the item splitter. Items can only be put into an item splitter using hoppers.
