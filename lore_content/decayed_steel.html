Obtaining

Decayed steel can be obtained using any tool, although hoes mine it fastest. It naturally generates in the rotting spires found in the End. If mined with a tool without Silk Touch, decayed steel has a 30% chance to drop 1-3 rotting flesh instead of itself. Each level of Fortune decreases the chance of rotting flesh drops.

Usage

Decayed steel is a gravity-affected block like sand or gravel, but it only falls when stepped on or if the block below is removed and not immediately if placed with an air block below.

Dispensers can place decayed steel when given a redstone signal if the dispenser is facing an air block.
