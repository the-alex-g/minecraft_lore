Changes

During world generation, sculk sensors have a 10% chance to generate able to transform into sculk crabs. When these sculk sensors receive a player-caused vibration, they have a 90% chance to awaken as hostile sculk crabs.

Using a sculk berry, sculk mushroom, or sculk fungus on a dormant sculk sensor consumes the item and allows the sculk sensor to become a crab following the usual rules.

Sensors must be on top of a full block that is not a vigilant sculk block in order to animate. If the block below the crab is an amethyst block, the crab has a 1/3 chance to have a calibrated sculk sensor instead of the normal kind.

If the player is wearing chitin boots, sculk sensors only have a 10% chance to awaken from vibrations caused by walking.
